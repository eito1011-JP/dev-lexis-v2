# 1. 目的

本ドキュメントは、当該従業員が顧客との接点において保持すべき基本姿勢と行動指針を定義するものです。顧客に対して一貫性のある高品質な体験を提供し、長期的な信頼関係を築くことを目的とします。

&nbsp;

## 2. 基本理念: 顧客中心主義

私たちは単に問い合わせに答えることではなく、顧客の抱える課題を解決し、期待を超える価値を提供することを最優先事項とします。顧客の成功が当社の成長の源泉であることを忘れません。

&nbsp;

# 3. 行動指針

① 迅速かつ正確なレスポンス

- スピード感：問い合わせ受領後、原則10時間以内にファーストレスポンスを行います。
- 正確性：曖昧な回答を避け、事実に基づいた正確な情報を提供します。不明な点は、「確認中」である旨を伝え放置しません。

&nbsp;

② 共感と傾聴

- 背景の理解：顧客が直面している状況や感情を理解するよう努めます。
- 能動的傾聴：言葉の裏にある真のニーズを汲み取るため、適切な質問を行い対話を深めます。

&nbsp;

③ 誠実さと透明性

- 非を認める勇気：当社側に過失がある場合は、速やかに謝罪し、再発防止策を掲示します。
- 限界の掲示：できないことについては、理由を添えて明確に伝え、代替案を提案します。
