# 1. 目的

本規定は、当社が提供するサービス及び商品において顧客から返金の要望が発生した際の判断基準と、適正かつ迅速な事務処理の手順を定めるものです。不適切な返金対応によるトラブルを防止し、顧客満足度の維持と適正な利益管理の両立を図ります。

&nbsp;

## 2. 返金の基本原則

原則として提供済みのサービス及び商品については返金に応じないものとします。ただし、以下の返金対象となるケースに該当する場合に限り本規定に基づき返金処理を行います。

&nbsp;

# 3. 返金対象となるケース

以下のいずれかに該当し、かつ購入契約から30日以内の期間は社内ルールに合わせて変更の申出があった場合を対象とします。

&nbsp;

- 商品・サービスの不備：提供した商品に物理的な欠陥がある、またはサービスが契約内容と著しく異なる場合。
- システムトラブル当社のシステムの不具合により二重決済が発生したまたはサービスが利用不能な状態が継続した場合。
- 解約条項に基づく場合、利用規約に定める中途解約条項に合致し、所定の解約手数料を差し引いた残額が発生する場合。
