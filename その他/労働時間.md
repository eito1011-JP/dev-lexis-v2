# 労働時間管理および勤務形態に関するガイドライン

## 1. 目的
本ドキュメントは、従業員の健康維持と生産性の向上を目的とし、労働時間の定義、適正な管理方法、および各種勤務制度の運用ルールを定めるものである。

## 2. 労働時間の定義
原則として、労働時間とは「会社の指揮命令下に置かれている時間」を指す。
* **実労働時間:** 業務に従事している時間（電話待機や手待時間を含む）。
* **休憩時間:** 労働から完全に解放されている時間。
* **時間外労働:** 所定労働時間を超えて行われる業務。

## 3. 標準的な勤務時間

## 4. 時間外・休日労働の原則
時間外労働（残業）は、業務上真にやむを得ない場合に限り、上司の事前承認を得て行うものとする。

1.  **36協定の遵守:** 法定労働時間を超える労働は、労使協定（36協定）の範囲内とする。
2.  **月間上限:** 原則として月45時間を超える時間外労働を禁止する。
3.  **深夜労働:** 22:00から翌5:00までの労働は、原則として緊急時を除き禁止する。

## 5. 休憩と休日
* **休憩:** 労働時間が6時間を超える場合は45分以上、8時間を超える場合は60分以上の休憩を付与する。
* **法定休日:** 毎週少なくとも1回、または4週間を通じて4日以上の休日を付与する。
* **勤務間インターバル:** 終業から翌日の始業まで、継続して11時間以上の休息時間を設けることを推奨する。
